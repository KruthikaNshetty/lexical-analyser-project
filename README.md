# lexical-analyser-project
## Description
- This project implements a simple lexical analyzer for C source code.  
- The lexical analyzer reads a C program character by character and identifies different tokens such as keywords, identifiers, operators, symbols, literals, and preprocessor directives.
- The system also ignores comments and separates valid lexemes into corresponding token types.

  ## 🚀 Features
  
- Detects C keywords
- Identifies identifiers
- Recognizes numeric constants
- Detects operators
- Detects symbols and brackets
- Handles string literals
- Handles character literals
- Detects preprocessor directives
- Ignores single-line comments
- Ignores multi-line comments

  ## 🛠 Technologies Used
  
- C Programming
- Compiler Design Concepts
- File Handling
- String Handling
- Character Classification Functions
- Command Line Arguments

  ## ⚙️ How It Works

- Reads C source file character by character
- Separates lexemes from source code
- Matches lexemes with predefined token patterns
- Identifies keywords using keyword table
- Detects operators and symbols
- Prints lexeme with corresponding token type
